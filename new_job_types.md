# Proposed New Industries & Job Types

## Current State

**12 industries, 102 job types** (industry ID 8 is missing/skipped)

Jobs frequently filed under "इतर (Other)" because no matching type exists — airport jobs, help desk, purchase executive, shift executive, etc.

## Proposed Changes to Existing Industries

### General (सामान्य) — Cleanup

The "General" industry has become a catch-all with 22 types. After creating new industries above, these types can be **moved** (not deleted):

| Current Type | Move To |
|-------------|---------|
| टेलिकॉलर (Telecaller) | Call Center / BPO |
| मेकॅनिक (Mechanic) | Automobile |
| शिपाई (Peon) | Keep in General |

This brings General down to ~20 types which is still fine.

### Migration Note
Existing jobs using "इतर (Other)" that match new types should be re-mapped after creating the new types. This can be done via a SQL update matching on description keywords. **No existing job data will be lost** — only the `job_type_id` foreign key would change.
